# Descrição do Projeto

O projeto "Trybe Warts" faz parte do currículo da formação de Desenvolvedor Web da [Trybe](https://www.betrybe.com/). Nesse projeto, os alunos têm a oportunidade de aplicar seus conhecimentos em HTML, CSS e JavaScript para criar um formulário inspirado em Harry Potter.

# Acesso ao projeto

Para acessar e executar este projeto, siga os passos abaixo:

1. Certifique-se de ter o Node.js e o npm instalados em sua máquina.

2. Clone o Projeto

```bash
git clone https://github.com/EversonDias/Trybe_Warts_with_HTML_Bootstrap_JavaScript.git TrybeWarts
```

3. Entre no projeto

```bash
cd TrybeWarts
```

4. inicie o projeto

```bash
npm start
```
